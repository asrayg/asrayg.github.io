The application utilizes image recognition technology, AI analysis, and access to veterinary resources to provide preliminary diagnoses and recommend appropriate actions. Coloring Minds Record System

The Coloring Minds Record Keeping System is a Java-based app for after school art classes. It manages records, attendance, fees, and payments efficiently. Features include email generation, balance management, and a user-friendly interface. It uses Java Swing for UI, Maven for email, and various data structures for record management. My mother, who runs Coloring Minds, extensively utilizes this system. Future plans include adding features like facial recognition for attendance, direct payment integration with Indian UPI servers, WhatsApp updates, and more.

WellGuard Card Generator

This script generates insurance cards with QR codes using Python libraries like PIL, qrcode, and reportlab. It reads customer information from a CSV file, then creates personalized cards with logos, text, and QR codes containing the data. The cards are saved as PNG files and compiled into a PDF document. The PDF's layout includes multiple cards per page. It offers flexibility in customization and scalability for generating a batch of insurance cards efficiently.
